Modelo Flexbox

Flexbox o Flexible Box es un módulo que provee una forma mucho más sencilla de ordenar, alinear y distribuir el espacio entre los objetos dentro de un contenedor, incluso cuando su tamaño es desconocido y/o dinámico.

La principal idea detrás de Flex es dar a los objetos dentro del contenedor la habilidad para alterar tanto el ancho como el alto de estos para que ocupen el máximo de espacio disponible y se acomoden a todo tipo de pantallas, expandiéndolos o encogiéndolos. Además, este está basado en una dimensión, es decir, todos los elementos se disponen en una línea, la cual los coloca de manera automática, haciendo lo anteriormente mencionado de encoger y expandir o poniendo unos debajo de otros cuando lleguen al tamaño máximo del contenedor si tenemos la propiedad llamada "flex-wrap" activa (aunque siga siendo "una línea" por así decirlo).

Lo más importante de Flexbox es que, a diferencia del tipo "block", verticalmente direccional, o el tipo "inline", horizontalmente direccional, el tipo Flex ignora las direcciones. Mientras los otros funcionan bien para páginas, tienen una gran falta de flexibilidad a la hora de respaldar aplicaciones grandes o complejas, especialmente cuando a cambios de orientación, redimensionar, estirar o encoger se refiere.

Nota: Flexbox es más indicado para los componentes de una aplicación y diseños pequeños, mientras que Grid es más indicado para diseños o layouts a mayor escala.

Ya que Flexbox es un módulo completo y no solo una propiedad, involucra varias cosas, incluyendo un set completo de propiedades. Algunas de ellas están hechas para ser colocadas en el contenedor o padre (flex container) mientras que otras van colocadas en los elementos interiores o hijos (flex items). Modelo Grid

CSS siempre se ha usado a la hora de organizar páginas, pero nunca ha sido muy bueno en esto. Primero, se utilizaban tablas, luego floats, posicionamiento y el "inline"-"block", no obstante todos estos métodos son en esencia trucos y dejan fuera muchas funcionalidades importantes (centrado vertical, por ejemplo). CSS Grid o Grid es y fue el primer modulo CSS creado específicamente para resolver los problemas de disposición que se han estado arreglando con estos trucos y que, comparado con cualquier otro sistema anterior, cambió completamente el modo en el que creamos interfaces web.

Grid es un sistema de disposición de elementos en dos dimensiones basado en cuadrículas y, aunque Flexbox es también una herramienta muy buena, su sistema unidireccional tiene casos de uso distintos (y de hecho funcionan muy bien juntos).

Antes de seguir, cabe destacar que Bootstrap tiene un sistema grid propio que, pese a ser muy similar al de CSS es también algo distinto, así que resumiré el de Bootstrap específicamente en esta página. Si quieres saber más sobre CSS Grid puedes encontrar información ilustrada en la web de CSS-Tricks (se abre en una pestaña nueva)
