Bootstrap

Para empezar con esta guía lo primero es saber de qué se trata Bootstrap. Bajo el signo "mobile-first", Bootstrap es un framework orientado a la creación de páginas web adaptables de forma rápida y sencilla, el cual añade elementos como iconos o clases que cuentan con distintos estilos.

Antes de proceder con el resto de la guía cabe mencionar que, pese a las ventajas de Bootstrap a la hora de crear páginas web como la rapidez a corto y largo plazo al tener gran parte de los estilos creados, estos cuentan con la etiqueta !important, y al ser tantos, es posible que nombremos una clase igual a una ya creada por Bootstrap, lo que da pie a errores o creer que nuestro css está mal y sin saber esto a perder tiempo buscando el fallo. Por suerte son fallos menores que se solucionan cambiando el nombre de nuestra clase o, dependiendo de como implementemos Bootstrap, editables directamente. Cabe destacar que Bootstrap no es necesario para hacer una página, como es el caso de esta, ni para darle estilo, ya que puedes hacer esto con css simplemente si así lo prefieres.

También decir que es recomendable entrar en los enlaces que encontrarás en esta página, ya que llevan directamente a las webs oficiales, donde está todo explicado en detalle.

Instalación y Uso

Bootstrap cuenta con una guía de instalación en su página principal (se abre en una pestaña nueva), a la cual acudiremos más adelante, aunque se resume en tres formas distintas dependiendo de nuestras preferencias.

jsDelivr:
La primera manera y más sencilla es el uso de jsDelivr, con la cual lo único que debemos hacer es añadir al head de nuestro HTML un link al CSS de Bootstrap y una o dos etiquetas script con referencias al JavaScript de Bootstrap y Popper dependiendo de que elijamos. Ambas etiquetas se pueden encontrar actualizadas en esta sección de su página de descarga (se abre en una pestaña nueva). Esta opción es la más conveniente, ya que tiene toda la funcionalidad de Bootstrap sin complicarnos demasiado y nos es útil si no queremos profundizar mucho en la estructura o queremos maquetar una página de un modo rápido.
CSS y JS Compilados:
La segunda opción más rápida es descargar el CSS y JS compilados directamente. Esto puede sernos útil, ya que al tener directamente la hoja de estilos que Bootstrap utiliza, podemos ver como funciona para así aprender a darle un mejor uso o incluso editarla para modificar el comportamiento de las distintas clases. Podemos encontrar el enlace de descarga al principio de la web de descargas (se abre en una pestaña nueva).
Gestor de paquetes:
Por último, podemos usar un gestor de paquetes a la hora de implementar Bootstrap en nuestra aplicación. Esta es la mejor opción en caso de que tengas una aplicación que emplee otro framework, extensa o compleja, para evitar las posibles incompatibilidades, aunque requiere del uso del compilador SASS y Autoprefixer, los cuales requiere Bootstrap para funcionar. En esta página (se abre en una pestaña nueva) se encuentran de forma más detallada que gestores tienen soporte y los comandos a usar.
