Bootstrap

Para empezar con esta guía lo primero es saber de que se trata Bootstrap. Bajo el signo "mobile-first", Bootstrap es un framework orientado a la creación de páginas web adaptables de forma rápida y sencilla, el cual añade elementos como iconos o clases que cuentan con distintos estilos.

Antes de proceder con el resto de la guía cabe mencionar que, pese a las ventajas de Bootstrap a la hora de crear páginas web como la rapidez a corto y largo plazo al tener gran parte de los estilos creados, estos cuentan con la etiqueta !important, y al ser tantos, es posible que nombremos una clase igual a una ya creada por Bootstrap, lo que da pie a errores o creer que nuestro css esta mal y sin saber esto a perder tiempo buscando el fallo. Por suerte son fallos menores que se solucionan cambiando el nombre de nuestra clase o, dependiendo de como implementemos Bootstrap, editables directamente. Cabe destacar que Bootstrap no es necesario para hacer una página, como es el caso de esta, ni para darle estilo, ya que puedes hacer esto con css simplemente si asi lo prefieres.

Instalación y Uso

Bootstrap cuenta con una guía de instalación en su página principal, a la cual acudiremos más adelante, aunque se resume en tres formas distintas dependiendo de nuestras preferencias.

jsDelivr:
La primera forma y más sencilla es el uso de jsDelivr, con la cual lo unico que debemos hacer es añadir al head de nuestro html un link al css de Bootstrap y una o dos etiquetas script con referencias al javascript de Bootstrap y Popper dependiendo de que elijamos. Ambas etiquetas se pueden encontrar actualizadas en esta sección de su página de descarga. Esta opción es la más conveniente, ya que tiene toda la funcionalidad de Bootstrap sin complicarnos demasiado y nos es util si no queremos profundizar mucho en la estructura o queremos maquetar una página de forma rápida.
CSS y JS Compilados:
La segunda opción más rápida es descargar el css y js compilados directamente. Esto puede sernos util ya que al tener directamente la hoja de estilos que Bootstrap utiliza, podemos ver como funciona para así aprender a darle un mejor uso o incluso editarla para modificar el comportamiento de las distintas clases. Podemos encontrar el enlace de descarga al principio de la web de descargas.
Gestor de paquetes:
Por último, podemos usar un gestor de paquetes a la hora de implementar Bootstrap en nuestra aplicación.Esta es la mejor opción en caso de que tengas una aplicación que use otro framework, extensa o compleja, para evitar las posibles incompatibilidades, aunque requiere del uso del compilador Sass y Autoprefixer, los cuales usa Bootstrap para funcionar. En esta página se encuentran de forma más detallada que gestores tienen soporte y los comandos a usar.

Breakpoints

Bootstrap está diseñado para ser responsive, por lo que gran cantidad de sus clases cuentan con la opción del uso de breakpoints. En resumidas cuentas podemos indicar que estilo queremos usar dependiendo del viewpoint, es decir, el tamaño de nuestra pantalla. Por ejemplo, si le damos a un párrafo las clases p-md-2 y p-lg-4, este tendrá un padding de 0.5 rem en pantallas con un ancho mayor o igual a 768px y 1.5 rem en pantallas un ancho mayor o igual a 992px. La estructura básica de estas clases se forma por un prefijo, que indica que estilo queremos aplicar como puede ser "p" para padding o "m" para margin, un interfijo, que denota el tamaño a partir del que funciona nuestra clase, y un sufijo, que sería el tamaño del prefijo de nuestra clase. A continuación dejare una tabla de interfijos para consultar los distintos breakpoints disponibles y otra de sufijos con los distintos tamaños soportados.

TIPO INTERFIJO TAMAÑO
Extra pequeña Ninguno <576px
Pequeña sm ≥576px
Mediana md ≥768px
Grande lg ≥992px
Extra grande xl ≥1200px
Extra extra grande xxl ≥1400px
SUFIJO USABLE EN VALOR
0 margin o padding 0
1 margin o padding $spacer * 0.25rem
2 margin o padding $spacer * 0.5rem
3 margin o padding $spacer

$spacer es un atributo editable que se encuentra en el Sass de Bootstrap en caso de que lo hayas instalado así. Tiene un valor por defecto de 1rem.
