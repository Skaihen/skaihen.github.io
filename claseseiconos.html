Clases

Bootstrap cuenta con varias clases con funcionalidades únicas, todas ellas explicadas en detalle en el apartado componentes de la web oficial de Bootstrap (se abre en una pestaña nueva), por lo que en esta sección encontraras una tabla ordenada alfabéticamente a modo de referencia para implementar dichas clases CSS de forma rápida, con posible ampliación en un futuro.

TIPO ESTILO USO VARIANTES APLICACIÓN
Affix Affix data-spy="affix"|data-offset-top=""   Affix es un plug-in que aplica la clase fixed a un elemento cuando llega a la altura especificada por el data-offset
Alertas Alert class="alert"   Esta clase indica que la etiqueta donde la pongas contiene una alerta. Por si sola solo añade un contenedor con margin y padding, por lo que es necesaria combinarla con una de las siguientes etiquetas de alerta para formar una alerta.
Alerta informativa class="alert-info"   Crea una caja celeste que indica información
Alerta de éxito class="alert-success"   Crea una caja verde que indica éxito o una acción positiva
Alerta de precaución class="alert-warning"   Crea una caja amarilla que indica precaución
Alerta de peligro class="alert-danger"   Crea una caja roja que indica peligro o una acción negativa
Link en alertas class="alert-link"   Se usa para que cualquier link dentro de una alerta tenga los mismos colores que esta
Alerta cerrable class="alert-dismissible"|<a href="#" class="close" data-dismiss="alert" aria-label="close">×</a> antes del contenido de la alerta   Indica que la alerta puede ser cerrada y crea un símbolo, en este caso una x, al final de la alerta para cerrarla
Desvanecimiento class="fade"   Añade un efecto de desvanecimiento a las alertas cerrables
Insignia Insignia class="badge"   Crea una insignia circular gris, normalmente usados como un indicador numérico
Botones Alert class="alert"   Esta clase indica que la etiqueta donde la pongas contiene una alerta. Por si sola solo añade un container con margin y padding, por lo que es necesaria combinarla con una de las siguientes etiquetas de alerta para formar una alerta.
Desvanecimiento class="fade"   Añade un efecto de desvanecimiento a las alertas cerrables
